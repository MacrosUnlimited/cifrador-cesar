# cifrador-cesar
Herramienta web interactiva para cifrar/descifrar textos mediante el cifrado César. Soporta el alfabeto español completo (incluyendo Ñ y vocales acentuadas), desplazamiento de 1 a 31 posiciones, opción para conservar espacios y signos de puntuación, sliders visuales y botones de copiado. Diseño moderno y responsive.
